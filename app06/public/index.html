Sobre o Projeto

Este projeto demonstra o uso do framework Express.js para criar uma API RESTful que consolida as funcionalidades dos projetos anteriores em uma única aplicação organizada e modular.

O Express.js é um framework web minimalista e flexível para Node.js que fornece um conjunto robusto de recursos para aplicações web e móveis.

Conceitos Abordados

  • Express.Router(): Organização de rotas em arquivos separados para melhor modularidade.
  • Middlewares: Uso de middlewares para processamento de requisições, logging, tratamento de erros e mais.
  • Estrutura de Rotas Separadas: Separação de responsabilidades com rotas organizadas por domínio.
  • Controladores: Implementação da lógica de negócio em controladores específicos para cada recurso.
  • Serviços Compartilhados: Reutilização de código através de serviços compartilhados entre diferentes partes da aplicação.

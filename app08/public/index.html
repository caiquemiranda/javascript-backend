API de Gerenciamento de Tarefas

Uma API RESTful para gerenciamento de tarefas com persistência em SQLite

Sobre

Esta API permite gerenciar tarefas, categorias e etiquetas através de endpoints RESTful. Utilize os endpoints abaixo para interagir com a API.

Desenvolvida com Node.js, Express e SQLite, seguindo os princípios da arquitetura MVC.

Funcionalidades

  • Gerenciamento completo de tarefas (CRUD)
  • Categorização de tarefas
  • Etiquetas para organização
  • Filtragem por prioridade, status e categoria
  • Busca por texto

Endpoints

Tarefas

GET /api/tasks Listar todas as tarefas
GET /api/tasks/:id Obter uma tarefa pelo ID
POST /api/tasks Criar uma nova tarefa
PUT /api/tasks/:id Atualizar uma tarefa
DELETE /api/tasks/:id Excluir uma tarefa
GET /api/tasks/category/:categoryId Listar tarefas por categoria
GET /api/tasks/label/:labelId Listar tarefas por etiqueta

Categorias

GET /api/categories Listar todas as categorias
GET /api/categories/:id Obter uma categoria pelo ID
POST /api/categories Criar uma nova categoria
PUT /api/categories/:id Atualizar uma categoria
DELETE /api/categories/:id Excluir uma categoria

Etiquetas

GET /api/labels Listar todas as etiquetas
GET /api/labels/:id Obter uma etiqueta pelo ID
POST /api/labels Criar uma nova etiqueta
PUT /api/labels/:id Atualizar uma etiqueta
DELETE /api/labels/:id Excluir uma etiqueta
GET /api/labels/task/:taskId Listar etiquetas de uma tarefa

Relacionamentos

POST /api/tasks/:taskId/labels/:labelId Adicionar uma etiqueta a uma tarefa
DELETE /api/tasks/:taskId/labels/:labelId Remover uma etiqueta de uma tarefa

Exemplo de Uso

Criar uma nova tarefa


fetch('/api/tasks', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        title: 'Estudar Node.js',
        description: 'Aprender sobre Express e SQLite',
        due_date: '2023-12-31',
        priority: 'alta',
        status: 'pendente',
        category_id: 3,
        labels: [1, 2]
    })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erro:', error));

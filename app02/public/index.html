Bem-vindo ao Servidor com Leitura de Arquivos

Este é um exemplo de servidor HTTP que lê arquivos do sistema de arquivos e os serve como páginas web.

Características deste projeto:

  • Utilizamos o módulo fs do Node.js para ler arquivos
  • Implementamos rotas para diferentes páginas
  • Servimos arquivos estáticos baseados em suas extensões
  • Tratamos erros e fornecemos páginas de erro personalizadas

Explore as diferentes páginas através do menu acima!

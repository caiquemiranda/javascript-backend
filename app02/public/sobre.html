Sobre o Projeto

Objetivo

Este projeto demonstra como criar um servidor HTTP em Node.js que pode ler e servir arquivos do sistema de arquivos local. É uma evolução do conceito de servidor básico, adicionando a capacidade de gerenciar conteúdo estático.

Recursos Técnicos

Este projeto utiliza os seguintes módulos nativos do Node.js:

  • http: Para criar o servidor web
  • fs: Para leitura e manipulação de arquivos
  • path: Para manipulação de caminhos de arquivo

Exemplos de Código

Aqui está um exemplo de como lemos arquivos de forma assíncrona:

function lerArquivo(caminhoArquivo) {
  return new Promise((resolve, reject) => {
    fs.readFile(caminhoArquivo, 'utf8', (err, conteudo) => {
      if (err) {
        reject(err);
        return;
      }
      resolve(conteudo);
    });
  });
}
